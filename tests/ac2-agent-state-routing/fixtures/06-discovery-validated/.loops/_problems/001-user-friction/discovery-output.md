# Discovery Output: 001 — User friction in onboarding

**Status:** validated
**Loop:** 1 — Discovery
**Last updated:** 2026-04-03

---

## Problem hypothesis

**Who:** New enterprise users setting up their account for the first time
**What:** Users abandon the onboarding process before completing account setup
**When/Where:** During the 5-step account setup flow; drop-off highest at step 3 when users must enter billing details before seeing any product value
**Why it matters:** Each abandoned setup delays activation; analysis shows 40% of abandoned setups never return, representing lost ARR
**How do you know:** Confirmed via 6 user interviews with enterprise users who recently attempted setup; all 6 independently described friction at the billing step as their reason for pausing or stopping
