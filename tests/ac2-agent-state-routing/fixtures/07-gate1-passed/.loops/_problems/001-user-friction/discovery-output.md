# Discovery Output: 001 — User friction in onboarding

**Status:** validated
**Loop:** 1 — Discovery
**Last updated:** 2026-04-03

---

## Problem hypothesis

**Who:** New enterprise users setting up their account for the first time
**What:** Users abandon the onboarding process before completing account setup
**When/Where:** Billing step (step 3 of 5) — users must enter billing details before seeing any product value
**Why it matters:** 40% of abandoned setups never return; direct activation ARR impact
**How do you know:** 6 external user interviews, March 2026. All independently cited the billing step.

---

## External validation

**Validated** — 6 external user interviews confirmed the problem.

---

## Sizing

**Reach:** ~50 new enterprise users per month
**Impact:** High — 40% abandonment, direct ARR impact
**Effort:** Medium — 3 screens, defer billing to post-activation
